## Tutorial

React官方网站教程。
使用到了React官方快速开发套件[create-react-app](https://github.com/facebookincubator/create-react-app),thanks React team;

## 使用方法

1. npm install (安装依赖)
2. npm start (开发)
3. npm run build (构建)
4. npm test (测试)
